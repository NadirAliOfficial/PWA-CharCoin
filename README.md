

# ⚡ CharCoin PWA – Frontend

This is the **Progressive Web App (PWA)** frontend for **CharCoin**, built with [Next.js](https://nextjs.org) and bootstrapped using [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app). It serves as the user interface for interacting with the CharCoin ecosystem, including staking, governance, rewards, and donation modules.

---

## 🚀 Getting Started

To run the development server locally:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

The app will hot-reload as you edit files. Main entry point: `app/page.tsx`.

---

## ✨ Features

* Progressive Web App (PWA) support with offline capabilities
* Clean UI powered by [Geist font](https://vercel.com/font)
* Dynamic integration with CharCoin smart contracts
* Modular architecture: staking, rewards, governance, and donations
* Mobile-optimized layout for Android and iOS homescreens
